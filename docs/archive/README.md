# Archive Directory

This directory contains archived versions of files that are no longer actively used but kept for reference.

## Contents

### `taxonomy-versions/`
- **taxonomy-v1.1.json** - Version 1.1 taxonomy (identical to current taxonomy.json, archived for reference)
- **taxonomy-v1.0.json.backup** - Backup of version 1.0 taxonomy

**Note:** Current active taxonomy is `src/data/taxonomy.json` (version 1.1.0)

---

## Why Archived?

These files are kept for:
- Historical reference
- Version comparison
- Rollback capability if needed

They are not imported or used by the application.
