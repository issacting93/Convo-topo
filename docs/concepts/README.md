# Conceptual and Theoretical Documentation

This folder contains theoretical frameworks, concepts, and core hypotheses underlying Conversational Cartography.

## Files

- **AXIS_DEFINITIONS.md** - Definitions and calculations for X/Y/Z axes
- **DESTINATION_VS_JOURNEY_HYPOTHESIS.md** - Core hypothesis: roles = destination, terrain = journey
- **RESEARCH_HYPOTHESIS_VALIDATION.md** - Validation of research hypotheses with examples
- **TERRAIN_DRIFTS_EXPLAINED.md** - Explanation of path generation algorithm
- **TERRAIN_METAPHOR_CRITIQUE.md** - Critical analysis of the terrain metaphor
- **TERRAIN_VS_CLASSIFIER_INDEPENDENCE.md** - Analysis of how terrain visualization differs from classification
- **SOCIOLINGUISTIC_TERMS.md** - Glossary of sociolinguistic terminology
- **ANOMALY_DETECTION_THESIS.md** - Thesis on anomaly detection as a third type of temporal information

## Purpose

These documents provide:
- Theoretical foundations (Social Role Theory, PAD model, etc.)
- Core hypotheses and their validation
- Conceptual frameworks for understanding the system
- Critical perspectives on methodology and design choices
